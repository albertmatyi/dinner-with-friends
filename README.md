# Git flow

1. `git clone ....`
1. `git checkout -b new-feature`
1. **work work**
1. `git add .`
1. `git commit -a`
1. `git push -u origin new-feature`
1. `git push` *n

1. Go to github
1. Create pull request
1. Merge pull request (review)

1. `git checkout mater`
1. `git pull`
1. `git checkout currently-working-on-feature`
1. `git rebase master` - apply
